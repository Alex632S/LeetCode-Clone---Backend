# LeetCode CMS - Angular Frontend

Система управления контентом для платформы программирования LeetCode Clone, построенная на Angular с использованием Angular Material.

## Функциональности

### Аутентификация
- **Вход в систему** - авторизация пользователей
- **Регистрация** - создание новых аккаунтов
- **JWT токены** - безопасная аутентификация
- **Защита маршрутов** - guard для админ-панели

### Управление задачами
- **Список задач** - просмотр всех задач с фильтрацией
- **Детали задачи** - полная информация о задаче
- **Создание/редактирование** - форма для управления задачами
- **Удаление задач** - с подтверждением действия

### Управление пользователями
- **Список пользователей** - просмотр всех пользователей
- **Редактирование ролей** - изменение прав доступа
- **Админ-панель** - управление пользователями системы

## Стэк

- **Angular**
- **Angular Material**
- **TypeScript**
- **RxJS**
- **Standalone Components**
- **JWT Authentication**
