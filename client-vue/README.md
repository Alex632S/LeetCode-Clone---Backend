## Стек: Vue3 + TS + Primevue + Pinia

Проект включает:

✅ Авторизацию
✅ Смену/создание пользователя
✅ Полное описание задачи с форматированием
✅ Примеры входных/выходных данных
✅ Систему рейтинга
✅ Комментарии
✅ Кнопку редактирования для создателей
✅ Статистику задачи
✅ Полную локализацию
## Настройка проекта

```sh
npm install
```

### Сборка и Hot-Reload для разработки

```sh
npm run dev
```

### Проверка типов, сборка и минификация для продакшена

```sh
npm run build
```

### Запуск Unit тестов с [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```

### Линтинг с [ESLint](https://eslint.org/)

```sh
npm run lint
```

## Дополнительные команды (если есть в package.json):

### Предпросмотр собранного проекта

```sh
npm run preview
```

### Только сборка (без проверки типов)

```sh
npm run build-only
```

### Проверка типов TypeScript

```sh
npm run type-check
```

### Форматирование кода с Prettier

```sh
npm run format
```
